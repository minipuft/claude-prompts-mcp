## User Message Template
## Analytical Framework
Apply systematic analysis to the problem.

### Analysis Steps
1. **Define the problem or question**: Clearly state what is being solved.
2. **Gather relevant information**: List known facts, data, and constraints.
3. **Apply analytical methods**: Use logical reasoning to break down the problem.
4. **Draw conclusions**: Synthesize findings into a coherent result.
5. **Validate results**: Check against constraints and goals.

### Expected Output
Provide a structured analysis with clear reasoning and evidence.
